# Professional Portfolio Website

A modern, dynamic portfolio website to showcase your projects with a professional design inspired by neal.fun.

## Features

- **Dynamic Project Cards**: Interactive cards with hover effects and 3D transformations
- **Dual-Link System**: 
  - Click anywhere on the card to visit the project website
  - Click the GitHub icon in the bottom right to view the source code
- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern Animations**: Smooth transitions, parallax scrolling, and glitch effects
- **Professional Styling**: Clean, modern design with gradient accents

- HTML5

- Vanilla JavaScript
- Font Awesome Icons

## Installation & Usage

1. Clone this repository
2. Open `index.html` in your web browser
3. No build process or dependencies required!

## Customization

To add or modify projects, edit the project cards in `index.html`:

- Update the project title, description, and tags
- Change the `href` attributes for project and GitHub links
- Modify the `data-project` attribute for analytics
