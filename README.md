# Professional Portfolio Website

A modern, dynamic portfolio website to showcase projects with bold motion, gradients, and interactive cards.

## Features

- **Interactive Project Cards**: Clickable cards with 3D tilt and hover lift
- **Dual-Link System**:
   - Click anywhere on the card to visit the project website
   - Click the GitHub icon to view source code
- **Motion-First UI**: Staggered reveals, parallax hero, glitch title, and ripple clicks
- **Responsive Layout**: Auto-fit grid scales cleanly across devices
- **Gradient-Driven Styling**: Vivid gradients, soft shadows, and neon accents

- HTML5

- Vanilla JavaScript (DOM effects and interactions)
- Font Awesome Icons

## Installation & Usage

1. Clone this repository
2. Open `index.html` in your web browser
3. No build process or dependencies required

## Customization

To add or modify projects, edit the project cards in `index.html`:

- Update the project title, description, and tags
- Change the `href` attributes for project and GitHub links
- Modify the `data-project` attribute for analytics/logging

To tweak the look and feel:

- Adjust theme colors in the `:root` section of `styles.css`
- Change motion timings in `script.js` and the CSS keyframes
